void Blocks::Draw_Blocks ( int  arr[SIZE_ROWS][SIZE_COLS])

Эта функция метод класса Blocks, рисует набор разноцветных блоков.

Кроме переменных своей структуры, в качестве параметра передается двумерный массив arr[][]

Аргументы
xx - координата верхнего левого угла блока
yy - координата верхнего левого угла блока
widthширина блока
heightвысота блока
GapXрасстояние до левого верхнего угла следующего в ряду блока
GapYрасстояние до левого верхнего угла следующего в столбце блока
