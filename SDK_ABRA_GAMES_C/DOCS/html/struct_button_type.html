Эта функция метод класса ButtonType, возвращает "1" при нажатии на кнопку

Принимает в качестве параметров текущие координаты мышки

Заметки
Вспомогательная функция, проверяющая нажатие мышки в ограниченную кнопкой площадь
Аргументы
xx - координата верхнего левого угла нарисованной кнопки
yy - координата верхнего левого угла нарисованной кнопки
mouse_xx - текущая координата мышки
mouse_yy - текущая координата мышки
dis_xкритическая разность по x, между началом нарисованной кнопки и текущей координатой x мышки
dis_yкритическая разность по y, между началом нарисованной кнопки и текущей координатой y мышки
Пример использования:
if (Pause_icon.ButtonCheckClick (txMouseX(), txMouseY()) == true)
{
(*ball).vx = 0;
(*ball).vy = 0;
}
